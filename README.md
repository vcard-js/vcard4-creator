A [vCard 4](https://datatracker.ietf.org/doc/html/rfc6350) creation library.

Install
-------
```sh-session
npm install @vcard/vcard4-creator
```

Usage
-----
```js
import vcard4Creator from '@vcard/vcard4-creator';
```

Testing
-------
```sh-session
npm test
```
